# App Icon

Drop the production app icon here as:

- `icon.ico` — Windows app/window icon (multi-size: 16/32/48/256)
- `icon.png` — 512x512 source, used to generate the splash screen mark and
  the in-app About screen logo

No icon is bundled by default in this scaffold. `frontend/public/` also
expects a copy for the browser-tab favicon during `npm run dev`.

Suggested direction (matches the CAST design system): a simple geometric
mark — a rounded shield or a stylized "cast net" line icon — in muted teal
on warm charcoal, no gradients, no gloss.
