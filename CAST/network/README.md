# Reserved — Future VISTA Integration

This folder is intentionally empty of logic.

It exists only as an extension point so that a future version of CAST can
optionally read traffic/threat context from **VISTA** (Project 1 of the
Cybersecurity Software Suite) without restructuring the app.

**Do not implement anything here yet.** No APIs, no network calls, no
imports from VISTA. CAST must continue to work fully offline and
standalone.
