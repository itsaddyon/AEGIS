# Reserved — Future CAST Integration

This folder is intentionally empty of logic.

It exists only as an extension point so that a future version of ARGUS
can optionally read a user's security-awareness completion/proficiency
data from **CAST** (Project 2 of the AEGIS Suite — Cyber Awareness &
Security Training) — for example, to contextualize how a Case File's
"educational mode" explanation is pitched based on what the user has
already learned.

**Do not implement anything here yet.** No APIs, no shared database,
no imports from CAST. ARGUS's Case File lessons must continue to be
fully self-contained and useful without CAST installed.
