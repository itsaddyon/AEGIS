The AEGIS Ecosystem

AEGIS acts as the centralized control center and command HUD for a modular suite of standalone cybersecurity tools. It provides users with single-click process orchestration (initializing or terminating subsystems) while constantly polling runtime states and real-time CPU/RAM hardware statistics.

All database instances and persistent configuration files for the modules are centralized under the user's localized appdata folder (%LOCALAPPDATA%/AEGIS/) to guarantee high-speed, permissions-isolated file access on Windows operating systems.

ARGUS — Advanced Real-time Guardian for Unified Security

ARGUS is the security suite's active intrusion detection system (NIDS) and threat logging console. It monitors local network adapters using raw socket capture or WinPcap interfaces, sniffing incoming frames for indicators of compromise (IoC).

What you get:

  • Deep Packet Inspection (DPI): Uses Scapy to parse packet headers and layer payloads (TCP, UDP, ICMP, DNS).
  • Custom Heuristic Detection Rules: Parses snort-like rules to flag malicious activities (e.g., rapid connection attempts indicating port scans, repeated SSH/HTTP errors indicating brute force).
  • Case File System: Dynamically compiles flagged incidents into readable security case files, scoring threat severity levels.
  • Interface Configuration: Provides list and filter controls to sniff on selected network interfaces.

CAST — Cyber Awareness Simulation Trainer

CAST serves as the suite's educational and training node. It presents users with interactive, gamified simulation labs designed to simulate standard security situations and teach modern SecOps principles.

What you get:

  • SecOps Practical Labs: Modules covering realistic defense mechanics, phishing header analysis, password entropy audits, and encryption configurations.
  • Gamification Engine: Tracks XP gains, achievements, and security ranks as you progress through training modules.
  • Verifiable Certifications: Awards digitally signed Certificates of Competency with unique verification keys to prove course completion.

VISTA — Visualized Information on Security & Traffic Analysis

VISTA is the packet traffic visualization and protocol analysis engine. It provides high-fidelity diagnostic dashboards detailing data flows across the system's active adapters.

What you get:

  • Bandwidth Analytics: Visualizes live throughput charts displaying data rate history by protocol.
  • Geographic Routing Map: Automatically performs IP geolocation lookups to show where external connection endpoints are physically located on a global layout.
  • DDoS Spike Analysis: Flags packet volume spikes and maps active sockets to assist with traffic diagnosis and mitigation planning.
